# StackSave AI

AI spend optimization platform for startups and engineering teams.

## Features

- AI spend audit
- Savings recommendations
- Dynamic analytics dashboard
- Supabase lead storage
- Recharts visualizations
- Responsive modern UI

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Supabase
- Recharts
- Vercel

## Setup

npm install
npm run dev
